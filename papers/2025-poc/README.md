# The power of context: Random Forest classification of near synonyms. A case study in Modern Hindi 

Jacek Bąkowski, Institute of Polish Language, Polish Academy of Sciences, Kraków, Poland.<br />

This directory contains code and data for the following paper:
The power of context: Random Forest classification of near synonyms. A case study in Modern Hindi
<br />
The code repository includes full reproduction data and code for all tables and figures in the paper, as well as various supplemental figures and explanations.
The full paper is available here:

## Linguistic data
The file vocab_large.py contains a list of synonyms in hindi.

## Word2Vec
The folder models/hindi contains the Word2Vec model used for this research.
For copyright reasons, the Hitenten model is not shared publicly.

## Python Notebook
The file The-Power-of-Context.ipynb contains a Jupyter Notebook to reproduce all the results described in the paper.
All the operations were performed with Python v3.11.6.
